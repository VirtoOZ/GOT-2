Это первый Pet проект на React.

Задачи проекта:
	1. Изучить возможности работы фронтенд части web-сайта с сервером,
 		в частности с API AnApiOfIceAndFire;
	2. Создать SPA;
	3. Посмотреть работу паттернов react;
	4. Применить react-drill-components для проброски данных между компонентами,
	а также поработать с state, props; 
	5. Написать SPA на классовых компонентах;
	6. Использовать методы жизненного цикла (в рамках классовых компонентов);
	7. Провести рефакторинг и перейти на функциональные компоненты;
	8. Изучить Хуки (в рамках функциональных компонентов);

Стек: React(v18.3.1), React Dom, React Router(v7.6.3);

Описание работы SPA:
Сайт получает данные от API по ЛОРу вселенной "Game of Trones" https://www.anapioficeandfire.com.
Состоит из 3х основных визуальных компонентов:
	1. Блок рандомного персонажа; 
	2. Блок список персонажей; 
	3. Детальные данные по выбранному персонажу из выбранного из списка.
Header включает в себя кнопки по выбору категории информации:
 Персонажи(Characters), Книги(Books), Дома(Houses);
При выборе элемента из списка происходит роут на страницу с деатльными данными данного элемента.

Примечание: 
Сервер плохо работает предположительно из-за расположения и политических решений.
Для корректной работы в России, нужно подключаться через VPN.
